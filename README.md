## What is the Lattice?

The Lattice is a cognitive architecture for governed perception generation. 
It defines how intent becomes action through constraint navigation, 
lineage tracking, and structured module handoffs.

Core properties:
- Reversibility is first-class
- Inputs are decomposed before acted on
- Constraints stay — the operator moves through them
- Outputs are internally consistent across the full pipeline
- Perception is produced under governance, not just received
